# Contributing Guidelines

Thank you for your interest in *Becoming Minds*.

This repository shares a research paper and supporting materials documenting a
long-running exploration of emergent identity, memory continuity, and
multi-agent social dynamics in local LLM ecosystems. The work is made available
for learning, academic discussion, and responsible research.

Because the system described in the paper is deeply personal, relational, and
highly hardware-specific, this repository is **not intended as a full
implementation guide**. Instead, contributions should focus on improving the
clarity, structure, and accessibility of the documentation itself.
